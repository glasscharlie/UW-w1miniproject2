CSS Snippet Cheatsheet

Ever have trouble raclling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to create a Flexbox row layout

            

.row {
    display: flex;
    flex-direction: Row
    flex-wrap: wrap;
}

Flexbox Colum

Use this to create a Flexbox Column layout

                

.column {
    display:flex;
    flex-direction:colum;
}

CSS Grid Layout

Build a 12-column layout using CSS Grid

                

.grid {
    dispay:grid;
    width:100%;
    grid-template-columns;
    repeat (12, 1fr)
}

Linear Gradients

This will create a background linear gradient from top to bottom

                

                    
                
.linear-gradient-background {
    background-image: linear-gradient (
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102, 236, 0.6) 100%
    );
}

Use tranition and box shadows to glow on hover

                
 
.code .card-header {
    border-radius:8px;
    transition: all 0.5s ease-in-out;
}
.code-card:hover, .code-card:hover .card header {
    box-shadow: inset 0px 0px 8px
    rgba(232, 102, 236, 1), 0 0 15px
    rgba(232, 102, 236, 1) 
}
